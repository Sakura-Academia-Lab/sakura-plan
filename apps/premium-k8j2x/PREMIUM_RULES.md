# Sakura Academia Premium 開発ルール

このディレクトリ (`apps/premium-k8j2x/`) 内でアプリやツールを開発・修正する際は、以下のルールを必ず遵守してください。

## 1. プロ仕様のクオリティ

- 提供するアプリはすべて「プロ仕様」であることを前提とします。
- ユーザーが課金するに値する高い質（デザイン、機能、安定性、UX）を追求してください。

## 2. PWA (Progressive Web App) 化の必須

- プレミアム版のツールは必ず **PWA化** してください。
- オフライン動作、ホーム画面への追加、フルスクリーン表示など、アプリに近い体験を提供します。

## 3. 開発フロー：プレミアム優先

- **新規ツールの開発も、まずこのディレクトリから開始してください**。
- プレミアム版の公開ページにアクセスできる課金ユーザーは、開発途中の最新ツールにいち早く触れることができるというメリットを提供します。
- プレミアム版で十分に検証・完成された後に、その機能を制限・簡略化したものを `apps/` 直下の同名ツールに反映します。
- **注意点**: `apps/` 直下に展開する簡略版では、PWA機能は含めない（外す）ようにしてください。

## 4. バージョン管理の徹底

- アプリを更新した際は、**毎回必ずバージョンの数値を更新**してください。
- `index.html` 内の表示（例: `Ver 1.0`）だけでなく、スクリプトやCSSの読み込みパラメータ（例: `app.js?v=1.1`）なども更新し、キャッシュによるトラブルを防ぎます。
- 更新内容は `UPDATE_LOG.md` に正確に記録してください。
- 同時に、**プレミアムポータルのトップページ (`apps/premium-k8j2x/index.html`) の更新情報欄も更新**してください。最新3件を表示し、それ以前のものは「過去の更新」へ移動させます。

---

> [!IMPORTANT]
> 「まずプレミアムで最高のものを。その後に標準版へ」が基本原則です。
