# 【更新告知】立体図形スライサー Ver 1.3 🚀

3Dで立体切断をシミュレーションできる「立体図形スライサー」をVer 1.3にアップデートしました！

今回の主な更新内容：
✅ **特定の頂点が選択できなかった不具合を解消**
   正八面体や球の極点（上下の端）など、一部の図形でスナップ点が反応しなかった問題を修正しました。
   これにより、より多彩な角度での切断シミュレーションが可能になります。

ぜひ、中学受験の立体図形学習にお役立てください！
