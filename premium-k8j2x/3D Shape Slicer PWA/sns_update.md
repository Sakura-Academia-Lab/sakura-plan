# 【更新告知】立体図形スライサー Ver 1.4 🚀

3Dで立体切断をシミュレーションできる「立体図形スライサー」をVer 1.4にアップデートしました！

今回の主な更新内容：
✅ **切断点のドラッグ移動機能を追加**
   一度選んだ切断点をクリック（タップ）したまま辺に沿って動かせるようになりました。直感的に「もう少しこっちかな？」といった調整が可能です。
✅ **操作性・判定の向上**
   スナップ点が少し掴みやすくなり、頂点（角）への移動もスムーズに行えるよう改善しました。
