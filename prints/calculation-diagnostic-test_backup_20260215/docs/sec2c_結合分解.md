## 単元の目的

25×4=100、125×8=1000などの黄金ペアを見つける！数を分解・結合して計算を簡単にする技法。

### 教育的意義

- **結合法則の活用**: (a×b)×c = a×(b×c)
- **黄金ペアの発見**: 計算の途中で25×4, 125×8を作る
- **数の分解**: 32=8×4に分解する視力
- **認知負荷の削減**: 複雑な計算を瞬時に解決

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **黄金ペアの暗記**: 25×4=100, 125×8=1000, 16×625=10000
- **反復の徹底**: 同じ問題を繰り返し出題してOK
- **基本の完全習得**: 瞬時に答えられる状態

### 問題例

```
25 × 4 → 100（黄金ペア！）
125 × 8 → 1000（黄金ペア！）
4 × 25 → 100（順序を変えても同じ）
8 × 125 → 1000
16 × 625 → 10000（第3の黄金ペア！）
```

### 注意事項

- ✅ 25×4=100, 125×8=1000, 16×625=10000を暗記
- ✅ 順序を変えても同じ（交換法則）
- ✅ 「黄金ペア」と明記

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **黄金ペアの倍数**: 25×8=200（=25×4×2）
- **パターン網羅**: 様々な倍数での練習
- **50×4, 250×4など**: 黄金ペアの変形

### 問題例

```
25 × 8 → 25×4×2 = 200
25 × 12 → 25×4×3 = 300
125 × 16 → 125×8×2 = 2000
125 × 24 → 125×8×3 = 3000
50 × 4 → 25×2×4 = 200
250 × 4 → 25×10×4 = 1000
```

### 注意事項

- ✅ 黄金ペアの倍数パターン
- ✅ 50=25×2、250=25×10など
- ✅ 様々な数で練習

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **数を分解**: 32×25 → 8×4×25 → 8×100 = 800
- **黄金ペアを作る**: 32を8×4に分解
- **瞬発力**: 瞬時に分解パターンを見抜く

### 問題例

```
32 × 25 → 8×4×25 = 8×100 = 800
48 × 25 → 12×4×25 = 12×100 = 1200
64 × 125 → 8×8×125 = 8×1000 = 8000
96 × 125 → 12×8×125 = 12×1000 = 12000
16 × 25 → 4×4×25 = 4×100 = 400
```

### 注意事項

- ✅ 数の分解パターンを明記
- ✅ 32=8×4、48=12×4、64=8×8など
- ✅ 瞬時に分解できるよう練習

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **結合法則**: 8×37×125 → (8×125)×37 = 1000×37 = 37000
- **順序の変更**: 掛け算の順序を変えて黄金ペアを作る
- **実践的応用**: 様々な場面で活用

### 問題例

```
8 × 37 × 125 → (8×125)×37 = 1000×37 = 37000
4 × 56 × 25 → (4×25)×56 = 100×56 = 5600
16 × 23 × 625 → (16×625)×23 = 10000×23 = 230000
125 × 48 × 8 → (125×8)×48 = 1000×48 = 48000
5 × 34 × 20 → (5×20)×34 = 100×34 = 3400
```

### 注意事項

- ✅ 結合法則を使って順序を変更
- ✅ 括弧を明示して黄金ペアを作る
- ✅ 様々なパターン

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **多項結合**: 16×125×5×2 → (16×125)×(5×2)
- **複数の黄金ペア**: 複数の黄金ペアを組み合わせる
- **Sランク向け**: 最高難度の問題

### 問題例

```
16×125×5×2 → (16×125)×(5×2) = 2000×10 = 20000
8×125×25×4 → (8×125)×(25×4) = 1000×100 = 100000
32×25×125×8 → (32×25)×(125×8) = 800×1000 = 800000
4×25×16×625 → (4×25)×(16×625) = 100×10000 = 1000000
2×5×5×4 → (2×5)×(5×4) = 10×20 = 200
```

### 注意事項

- ✅ 複数の黄金ペアを同時に作る
- ✅ 括弧の使い方に注意
- ✅ 大きな数の計算

## 避けるべきパターン

### ❌ 黄金ペアが作れない数

```
NG: 23 × 37
理由: 黄金ペアが作れない。本単元の趣旨に合わない。
```

### ❌ 割り算の混在

```
NG: 32 × 25 ÷ 4
理由: 本単元は掛け算に限定。割り算は別単元。
```
