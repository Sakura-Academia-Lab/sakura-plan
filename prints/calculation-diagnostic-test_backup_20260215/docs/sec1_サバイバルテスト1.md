# 【第1部】反射神経 サバイバルテスト1 設計書

**ファイル**: `sec1_survival1.js`
**ステージ**: 第1部：反射神経
**テスト形式**: 連続正解チャレンジ 40問

---

## テストの目的

第1部「反射神経」の単元A～D（小数分数変換、四角数三角数、円周率、素数判定倍数判定）から各パターン・レベル1問ずつ計40問を出題し、基本的な計算の自動化レベルを測定する。

### テスト形式

- **問題数**: 40問（4単元 × 各10問）
- **制限時間**: 3分
- **採点方法**: 番号順に解き、連続正解数が得点（Q1から何問連続で正解したか）
- **セット数**: 5セット（パターンA, B, C, D, E）

### 判定基準

| 得点 | ランク | 状態 |
|------|--------|------|
| 35～40問 | S | 完全自動化（Mastery / Automatic） |
| 25～34問 | A | 流暢性（Fluency） |
| 15～24問 | B | 手順的知識（Procedural） |
| 0～14問 | C | 未習熟（Pre-structural） |

## 設計上の注意点

### ○×問題のバランス

各パターンで、素数判定（パ1）と倍数判定（パ2）の○×問題を以下のようにバランス配置：

**パターンA**:
- Level 1: ○, ○
- Level 2: ○, ×
- Level 3: ×, ○
- Level 4: ×, ○
- Level 5: ×, ○

合計：○5個、×5個（完全バランス）

### 答えの重複回避

同じテスト内で、逆の変換問題が近接しないよう配慮：
- ✅ Q1: 0.5→1/2、Q2: 1/4→0.25（異なる数値）
- ❌ Q1: 0.5→1/2、Q2: 1/2→0.5（逆問題、NG）

同じレベル内（Q1-Q2, Q9-Q10など）では、異なる数値を使用。

### 前問からの類推防止

- 単元Cの連続問題（Q5-Q6など）は異なる数値を使用
- 計算結果が前の問題のヒントにならないよう配慮
