# サバイバルテスト 用紙レイアウト設計書

## 概要

この文書は、サバイバルテスト（連続正解チャレンジ）の印刷用紙（PDF）の見た目・構成・レイアウトを定義します。

---

## 用紙仕様

| 項目 | 仕様 |
|------|------|
| 用紙サイズ | A4（210mm × 297mm） |
| 印刷面 | 両面印刷（表：問題、裏：解答・解説） |
| 余白 | 上下左右：10mm |
| フォント | Kiwi Maru（日本語）、システムフォント（数式） |

---

## 表面（問題ページ）構成

### ヘッダー部（上部）

```
┌─────────────────────────────────────────────────────┐
│ [ステージラベル]                          [印刷日付]  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│ [単元タイトル]【セットA/B/C】                   名前:  │
└─────────────────────────────────────────────────────┘
```

#### 要素詳細

| 要素 | 内容 | スタイル |
|------|------|----------|
| ステージラベル | 「ステージ1：反射神経」など | 12pt、太字、青色 |
| 印刷日付 | YYYY/M/D形式 | 10pt、グレー、右寄せ |
| 単元タイトル | 「反射神経【連続正解チャレンジ 35】」 | 18pt、太字、黒 |
| セットラベル | 「【セットA】」 | 16pt、太字、赤色 |
| 名前欄 | 「名前:」＋記入欄（下線） | 12pt、記入欄幅60mm |

### 目標・ルール説明部

```
┌─────────────────────────────────────────────────────┐
│ 目標：ルール：制限時間3分。番号順に解くこと！         │
│       1番から連続正解したところまでが得点             │
└─────────────────────────────────────────────────────┘
```

- 背景色：薄黄色（#fff9e6）
- 枠線：1px 実線 オレンジ
- パディング：8px
- フォントサイズ：11pt
- アイコン：⏱️（時計）

### 問題リスト部（2列レイアウト）

```
┌─────────────────┬─────────────────┐
│ (1) 問題式      │ (18) 問題式     │
│     [   ]       │      [   ]      │
│                 │                 │
│ (2) 問題式      │ (19) 問題式     │
│     [   ]       │      [   ]      │
│                 │                 │
│ ...             │ ...             │
│                 │                 │
│ (17) 問題式     │ (34) 問題式     │
│      [   ]      │      [   ]      │
└─────────────────┴─────────────────┘
```

#### レイアウト仕様

| 項目 | 仕様 |
|------|------|
| 列数 | 2列（左右均等） |
| 問題番号 | (1)～(35)、太字、12pt |
| 問題式 | 数式（MathJax）、14pt |
| 解答欄 | 幅40mm、高さ8mm、下線 |
| 問題間スペース | 6mm |
| 列間スペース | 15mm |

#### 特殊対応

- **素因数分解問題**: 解答欄を1.5倍に拡大（幅60mm）
- **○×問題**: 解答欄を縮小（幅20mm）

## 裏面（解答・解説ページ）構成

### ヘッダー部

```
┌─────────────────────────────────────────────────────┐
│ 解答と指導ポイント                                   │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━   │
│ [単元タイトル]【セットA/B/C】解説                     │
└─────────────────────────────────────────────────────┘
```

- 背景色：薄緑色（#e8f5e9）
- ステージラベルの代わりに「解答と指導ポイント」

### 解答・解説リスト（2列レイアウト）

```
┌─────────────────┬─────────────────┐
│ (1) 問題式      │ (18) 問題式     │
│ 答: 121         │  答: 1.21       │
│                 │                 │
│ (2) 問題式      │ (19) 問題式     │
│ 答: 144         │  答: 1.44       │
│                 │                 │
│ ...             │ ...             │
└─────────────────┴─────────────────┘
```

#### レイアウト仕様

| 項目 | 仕様 |
|------|------|
| 列数 | 2列 |
| 問題番号 | (1)～(35)、太字、11pt |
| 問題式 | 数式、12pt |
| 解答 | 「答:」＋数式、赤色、太字、14pt |
| 問題間スペース | 4mm（表より密） |

#### 注意事項

- **解説文（annotation）は非表示**: サバイバルテストは35問と多いため、解説文は省略し、問題と答えのみをコンパクトに配置
- **指導ポイントは別途記録用紙**: 詳細な解説や指導ポイントは「攻略パスポート」に記載

## デザイン原則

### 1. 視認性重視

- **問題番号は太字**: (1)、(2) など、視線誘導を強化
- **色分け**: ステージラベル（青）、セットラベル（赤）、解答（赤）
- **余白**: 問題間に十分なスペース（6mm以上）

### 2. 採点しやすさ

- **2列レイアウト**: 表裏を重ねて透かして採点できる配置
- **解答欄は大きめ**: 手書きで書きやすい高さ（8mm）

### 3. プリンター互換性

- **カラー不要**: 黒インクのみで印刷可能（色は補助）
- **余白**: 四辺10mm確保でプリンター誤差に対応

## CSS実装対応表

| 要素 | CSSクラス | 主要スタイル |
|------|-----------|-------------|
| 表面 | `.survival-sheet` | 2列レイアウト、`column-count: 2` |
| 問題リスト | `.problems` | リストスタイル無効化 |
| 問題項目 | `.problem-item` | `display: flex`、問題間スペース6mm |
| 問題番号 | `.p-num` | `font-weight: bold`、幅30px |
| 問題式 | `.p-content` | MathJax、14pt |
| 解答欄 | `.answer-box-empty` | `border-bottom: 1px solid`、幅40mm |
| 解答欄（広） | `.answer-box-empty.wide` | 幅60mm（素因数分解用） |
| 解答 | `.ans-box-filled` | 赤色、太字、14pt |

---

## レスポンシブ対応

### 印刷時（@media print）

- サイドバー非表示
- ページ分割最適化（`.page-break`）
- 余白調整（`@page { margin: 10mm; }`）

### 画面表示時

- サバイバルテストは2列表示を維持
- タブレット・スマホでは1列に変更（768px以下）
