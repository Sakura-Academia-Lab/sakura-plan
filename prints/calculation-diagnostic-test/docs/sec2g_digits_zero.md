## 単元の目的

10倍、100倍、1000倍は小数点を動かすだけ！桁数とゼロの構造を見抜く視力を鍛える。

### 教育的意義

- **小数点移動の理解**: ×10 → 小数点を右に1つ
- **黄金ペアとの融合**: 25×40 = 25×4×10
- **桁数の構造**: ゼロの個数と桁数の関係
- **認知負荷の削減**: 瞬時に桁数を判断

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **小数点移動**: 25×10 → 小数点を右に1つ → 250
- **ゼロの個数**: ×100 → ゼロ2個追加
- **基本の徹底**: 瞬時に答えられる状態

### 問題例

```
25×10 → 250（小数点を右に1つ）
34×10 → 340
12×100 → 1200（小数点を右に2つ）
25×100 → 2500
12×1000 → 12000（小数点を右に3つ）
```

### 注意事項

- ✅ 小数点移動のルールを明記
- ✅ ゼロの個数と小数点の移動を対応
- ✅ 様々な数で練習

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **割り算**: 250÷10 → 小数点を左に1つ → 25
- **逆操作**: 掛け算と割り算は逆の操作
- **小数も同じ**: 3.4×10 = 34

### 問題例

```
250÷10 → 25（小数点を左に1つ）
2500÷100 → 25（小数点を左に2つ）
12000÷1000 → 12（小数点を左に3つ）
3.4×10 → 34（小数も同じ）
0.78×1000 → 780
```

### 注意事項

- ✅ 割り算は小数点を左に
- ✅ 小数での計算も同じルール
- ✅ 様々なパターン

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **黄金ペアとの融合**: 25×40 = 25×4×10 = 1000
- **ゾロ目**: 11111×9 = 99999
- **構造の発見**: 複数の技法を組み合わせる

### 問題例

```
25×40 → 25×4×10 = 100×10 = 1000
25×400 → 25×4×100 = 10000
125×80 → 125×8×10 = 10000
33333-9999 → 23334
11111×9 → 99999
```

### 注意事項

- ✅ 黄金ペアとゼロ処理を組み合わせる
- ✅ ゾロ目のパターン（11111系列）
- ✅ 瞬発力を鍛える

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **小数どうしの積**: 2.5×0.4 = 25×4÷100 = 1
- **桁数の調整**: 整数に直して計算してから戻す
- **実践的応用**: 複雑な小数の計算

### 問題例

```
2.5×0.4 → 25×4÷100 = 100÷100 = 1
1.25×0.8 → 125×8÷1000 = 1
0.16×6.25 → 16×625÷10000 = 1
0.05×200 → 5×200÷100 = 10
```

### 注意事項

- ✅ 小数を整数に直す
- ✅ 桁数を調整して戻す
- ✅ 実践的な応用

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **ゾロ目処理**: 99999÷11111 = 9
- **パターンの発見**: 11111×n = nnnnn
- **Sランク向け**: 最高難度の問題

### 問題例

```
99999÷11111 → 9（11111×9=99999）
77777÷11111 → 7
55555÷11111 → 5
11111×7 → 77777
0.125×0.8 → 0.1
25×12 → 300（25×4×3）
```

### 注意事項

- ✅ ゾロ目のパターン認識
- ✅ 複雑な小数の計算
- ✅ 複数の技法の組み合わせ

## 避けるべきパターン

### ❌ 桁数が多すぎる

```
NG: 123456×1000000
理由: 桁数が多すぎる。小学生の範囲を超える。
```

### ❌ 複雑な小数

```
NG: 1.234×5.678
理由: 小数点の処理が複雑。基本的な小数に限定。
```
