## 単元の目的

補数（足して100, 1000, 1になる数）を瞬時に見つける力を鍛える。引き算は補数で解く技法を習得。

### 教育的意義

- **補数の概念**: 32+68=100、234+766=1000、0.375+0.625=1
- **引き算の高速化**: 100-47 → 47の補数は53
- **暗算の工夫**: 98+47 → (98+2)+45 = 100+45 = 145
- **認知負荷の削減**: 補数を瞬時に見つけることで、複雑な計算を簡略化

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **足して100**: 32+68=100、47+53=100など
- **桁ごとの補数**: 十の位3→7、一の位2→8
- **反復の徹底**: 同じ問題を繰り返し出題してOK

### 問題例

```
32 + □ = 100 → 68（3→7、2→8）
47 + □ = 100 → 53（4→5、7→3）
25 + □ = 100 → 75（2→7、5→5）
68 + □ = 100 → 32（6→3、8→2）
83 + □ = 100 → 17（8→1、3→7）
```

### 注意事項

- ✅ 桁ごとの補数ルール（十の位：a→9-a+1、一の位：b→10-b）
- ✅ 同じ問題を繰り返してOK
- ✅ 順序を入れ替えて出題（32+68、68+32）

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **足して1000**: 234+766=1000、348+652=1000など
- **3桁の補数**: 百の位2→7、十の位3→6、一の位4→6
- **パターン網羅**: 様々な3桁の数での練習

### 問題例

```
234 + □ = 1000 → 766
348 + □ = 1000 → 652
567 + □ = 1000 → 433
763 + □ = 1000 → 237
125 + □ = 1000 → 875
375 + □ = 1000 → 625
```

### 注意事項

- ✅ 3桁の補数も桁ごとに計算
- ✅ 125+875=1000など、きれいな数も出題
- ✅ Level 1の復習問題も混ぜる

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **引き算形式**: 100-47 → 47の補数は53
- **1000からの引き算**: 1000-348 → 348の補数は652
- **足して1**: 1-0.375=0.625、1-3/8=5/8
- **小数・分数の補数**: 0.375+0.625=1、3/8+5/8=1

### 問題例

```
100 - 47 → 53（47の補数）
1000 - 348 → 652（348の補数）
1 - 0.375 → 0.625（3/8の補数は5/8）
1 - 3/8 → 5/8
1 - 0.625 → 0.375（5/8の補数は3/8）
1 - 7/8 → 1/8
```

### 注意事項

- ✅ 引き算は補数で解く技法を明記
- ✅ 小数と分数の両方を出題
- ✅ 1/8系列（0.125, 0.375, 0.625, 0.875）を使用

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **補数を使った工夫**: 98+47 → (98+2)+45 = 100+45 = 145
- **999, 998との計算**: 998+234 → (998+2)+232 = 1000+232
- **実践的応用**: 様々な場面で補数を使う

### 問題例

```
98 + 47 → (98+2)+45 = 145
97 + 38 → (97+3)+35 = 135
96 + 85 → (96+4)+81 = 181
998 + 234 → (998+2)+232 = 1232
10000 - 2468 → 7532（2468の補数）
```

### 注意事項

- ✅ 98, 97, 96など100に近い数を使う
- ✅ 補数を使った工夫の手順を明記
- ✅ 10000からの引き算も出題

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **補数の連鎖**: (100-32)+(100-68)=68+32=100
- **複雑な計算**: 100-(47+32)など
- **99+99, 999+999**: (100-1)+(100-1)=200-2=198
- **Sランク向け**: 最高難度の問題

### 問題例

```
(100-32) + (100-68) → 68+32 = 100
(1000-234) + (1000-766) → 766+234 = 1000
100 - (47+32) → 100-79 = 21
1/8 + □ = 1（□は分数で） → 7/8
99 + 99 → (100-1)+(100-1) = 198
999 + 999 → (1000-1)+(1000-1) = 1998
```

### 注意事項

- ✅ 補数の連鎖計算
- ✅ 99+99, 999+999のパターン
- ✅ 分数での逆算

## 避けるべきパターン

### ❌ 複雑すぎる数

```
NG: 12345 + □ = 100000
理由: 桁数が多すぎる。100, 1000, 10000までに限定。
```

### ❌ 循環小数の補数

```
NG: 1 - 1/3 = 2/3 = 0.666...
理由: 循環小数は小学生には難しい。1/8, 1/4系列に限定。
```

### ❌ 負の数

```
NG: 32 + □ = 10 → -22
理由: 負の数は中学数学。正の数のみ。
```
