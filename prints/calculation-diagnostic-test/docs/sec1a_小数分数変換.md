# 【単元A】小数↔分数変換 設計書

**ファイル**: `sec1a_decimal_fraction.js`
**ステージ**: 第1部：反射神経
**単元ID**: `sec1a`

---

## 単元の目的

小数と分数の相互変換を「自動化」し、計算の基礎体力を構築する。0.5を見たら即座に1/2が浮かぶ状態（System 1：直感）を目指す。

---

## レベル構成

| レベル | 名称 | 問題数 | 文字通り | 主な内容 |
|--------|------|--------|----------|----------|
| Level 1 & 2 | | 22問 | 1分 | 0.5, 0.25, 0.75, 1/8系列, 1/5系列 |
| Level 3 | | 16問 | 1分30秒 | 帯分数（1.25, 1.5, 1.75など） |
| Level 4 | | 16問 | 2分 | 計算での活用（0.75×8, 0.5+1/4） |
| Level 5 | | 16問 | 2分30秒 | 割り算（÷0.25）、帯分数と小数の混在 |

## Level 1 & 2

### 設計意図

- **重複排除と網羅**: 頻出する基本小数を11パターンに絞り込み、重複なく網羅。
- **順序の工夫**: 左右の列で同じ番号が逆の答えにならないよう、右側の順序をシャッフル。

### 問題一覧

| No | パターン1（小数→分数） | パターン2（分数→小数） |
|:---:|:---|:---|
| 1 | 0.5 → 1/2 | 3/8 → 0.375 |
| 2 | 0.25 → 1/4 | 1/5 → 0.2 |
| 3 | 0.75 → 3/4 | 3/4 → 0.75 |
| 4 | 0.125 → 1/8 | 1/8 → 0.125 |
| 5 | 0.375 → 3/8 | 4/5 → 0.8 |
| 6 | 0.625 → 5/8 | 1/2 → 0.5 |
| 7 | 0.875 → 7/8 | 5/8 → 0.625 |
| 8 | 0.2 → 1/5 | 2/5 → 0.4 |
| 9 | 0.4 → 2/5 | 7/8 → 0.875 |
| 10 | 0.6 → 3/5 | 1/4 → 0.25 |
| 11 | 0.8 → 4/5 | 3/5 → 0.6 |

## Level 3

### 設計意図

- **帯分数で答える**: 1.25 を 1と1/4 で答える習慣をつける。
- **整数部分の分離**: 整数と小数部分を分けて認識する力を養う。

### 問題一覧

| No | パターン1（小数→帯分数） | パターン2（帯分数→小数） |
|:---:|:---|:---|
| 1 | 1.25 → 1と1/4 | 1と1/4 → 1.25 |
| 2 | 1.5 → 1と1/2 | 1と1/2 → 1.5 |
| 3 | 1.75 → 1と3/4 | 1と3/4 → 1.75 |
| 4 | 2.5 → 2と1/2 | 2と2/5 → 2.4 |
| 5 | 1.25 → 1と1/4 | 1と1/4 → 1.25 |
| 6 | 2.25 → 2と1/4 | 3と1/5 → 3.2 |
| 7 | 1.75 → 1と3/4 | 2と3/4 → 2.75 |
| 8 | 3.5 → 3と1/2 | 1と3/8 → 1.375 |

## Level 4

### 設計意図

- **実践的活用**: 小数を分数に、または分数を小数に変換することで計算が劇的に楽になるパターンを学習。

### 問題一覧

| No | パターン1（小数を分数にして×） | パターン2（小数と分数の混在＋ー） |
|:---:|:---|:---|
| 1 | 0.75 × 8 = 6 | 0.5 + 1/4 = 0.75 |
| 2 | 0.125 × 8 = 1 | 3/4 - 0.25 = 0.5 |
| 3 | 0.5 × 16 = 8 | 1/4 + 0.5 = 0.75 |
| 4 | 0.25 × 12 = 3 | 0.8 - 1/5 = 0.6 |
| 5 | 0.8 × 15 = 12 | 1/2 + 0.375 = 0.875 |
| 6 | 0.4 × 25 = 10 | 0.625 + 1/4 = 0.875 |
| 7 | 0.625 × 8 = 5 | 7/8 - 0.5 = 0.375 |
| 8 | 0.375 × 8 = 3 | 0.4 + 2/5 = 0.8 |

---

## Level 5

### 設計意図

- **割り算の変換**: ÷0.25=×4 などの「逆数の掛け算」を自動化。
- **複合計算**: 帯分数と小数が混在した計算を処理する。

### 問題一覧

| No | パターン1（小数の割り算） | パターン2（帯分数と小数の混在） |
|:---:|:---|:---|
| 1 | 5 ÷ 0.2 = 25 | 1と1/4 + 0.5 = 1.75 |
| 2 | 3 ÷ 0.25 = 12 | 2と1/2 - 0.75 = 1.75 |
| 3 | 7 ÷ 0.5 = 14 | 0.8 + 1と1/5 = 2.0 |
| 4 | 5 ÷ 0.125 = 40 | 1と3/4 - 0.25 = 1.5 |
| 5 | 8 ÷ 0.2 = 40 | 3と1/2 + 0.375 = 3.875 |
| 6 | 10 ÷ 0.25 = 40 | 2と1/4 - 0.5 = 1.75 |
| 7 | 9 ÷ 0.5 = 18 | 1と3/8 + 0.25 = 1.625 |
| 8 | 3 ÷ 0.125 = 24 | 3と4/5 - 0.6 = 3.2 |
