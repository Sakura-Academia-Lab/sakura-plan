## 単元の目的

25×4=100, 125×8=1000, 16×625=10000などの「黄金ペア」を暗記し、掛け算・割り算を瞬時に実行できる状態を目指す。

### 教育的意義

- **黄金ペアの暗記**: 25×4=100, 125×8=1000, 16×625=10000
- **変換技法**: ×25は÷4×100、×125は÷8×1000と変換
- **計算の高速化**: 32×25 → 32÷4×100 = 8×100 = 800
- **認知負荷の削減**: 暗記により、複雑な計算を瞬時に解決

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **基本の暗記**: 25×4=100, 125×8=1000
- **反復の徹底**: 同じ問題を繰り返し出題してOK
- **倍数パターン**: 25×8=200（=25×4×2）、25×12=300（=25×4×3）

### 問題例

```
25 × 4 → 100（黄金ペアの基本！）
125 × 8 → 1000（もう一つの黄金ペア！）
25 × 8 → 200（=25×4×2）
25 × 12 → 300（=25×4×3）
25 × 16 → 400（=25×4×4）
25 × 20 → 500（=25×4×5）
```

### 注意事項

- ✅ 25×4=100, 125×8=1000は「黄金ペア」と明記
- ✅ 倍数パターンを理解させる（25×8=25×4×2）
- ✅ 順序を変えても同じ（4×25=100）

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **÷4×100技法**: 32×25 → 32÷4=8 → 8×100=800
- **125の倍数**: 125×16=2000（=125×8×2）
- **パターン網羅**: 様々な数での計算

### 問題例

```
32 × 25 → 32÷4=8 → 8×100=800
48 × 25 → 48÷4=12 → 12×100=1200
125 × 16 → 125×8×2 = 2000
125 × 32 → 125×8×4 = 4000
```

### 注意事項

- ✅ ×25は÷4×100と変換する技法を明記
- ✅ ×125は÷8×1000と変換する技法を明記
- ✅ 様々な数で練習

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **逆算**: □×25=1200 → 1200÷100=12 → 12×4=48
- **625の導入**: 16×625=10000（第3の黄金ペア）
- **瞬発力**: 考える時間を与えない（自動化の確認）

### 問題例

```
□ × 25 = 1200 → □=48（1200÷100×4）
□ × 125 = 3000 → □=24（3000÷1000×8）
16 × 625 → 10000（黄金ペア！）
32 × 625 → 20000
128 × 25 → 3200（128÷4=32）
```

### 注意事項

- ✅ 逆算問題で理解を確認
- ✅ 16×625=10000を導入
- ✅ 大きな数（128, 144など）も出題

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **割り算への応用**: 3600÷25 → 3600÷100×4 = 36×4 = 144
- **実践的活用**: 様々な場面で黄金ペアを使う
- **構造視力**: 25, 125, 625を見つける力

### 問題例

```
3600 ÷ 25 → 3600÷100×4 = 36×4 = 144
5000 ÷ 125 → 5000÷1000×8 = 5×8 = 40
50000 ÷ 625 → 50000÷10000×16 = 5×16 = 80
240 × 25 → 240÷4×100 = 60×100 = 6000
```

### 注意事項

- ✅ 割り算も同じ変換技法（÷25 → ÷100×4）
- ✅ 大きな数での練習
- ✅ 実践的な応用

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **複合計算**: (48×25)+(16×125)など
- **平方数**: 25×25=625、125×125=15625
- **Sランク向け**: 最高難度の問題

### 問題例

```
(48×25) + (16×125) → 1200+2000 = 3200
(64×25) - (8×125) → 1600-1000 = 600
25 × 25 → 625
125 × 125 → 15625
□ × 25 × 4 = 10000 → □=100
```

### 注意事項

- ✅ 複数の黄金ペアを組み合わせる
- ✅ 25²=625、125²=15625も暗記
- ✅ 逆算の複雑なパターン

---

## 避けるべきパターン

### ❌ 割り切れない数

```
NG: 23 × 25 = 575
理由: 23は4で割り切れない。混乱を招く。
```

### ❌ 3125（=5⁵）など大きすぎる数

```
NG: 3125 × 32 = 100000
理由: 3125は小学生には難しすぎる。625までに限定。
```

### ❌ 小数の複雑な計算

```
NG: 0.125 × 625 = 78.125
理由: 小数点の処理が複雑。整数中心に。
```
