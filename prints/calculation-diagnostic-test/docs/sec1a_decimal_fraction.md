## 単元の目的

小数と分数の相互変換を「自動化」し、計算の基礎体力を構築する。0.5を見たら即座に1/2が浮かぶ状態（System 1：直感）を目指す。

### 教育的意義

- **暗記項目**: 基本的な変換（0.5=1/2、0.25=1/4など）は反復によって長期記憶化
- **認知負荷の削減**: 変換が自動化されれば、複雑な問題でもワーキングメモリを節約できる
- **応用への布石**: Level 4～5で計算での活用（0.5+1/4など）に繋げる

## レベル構成

| レベル | 名称 | 問題数 | 時間目標 | 主な内容 |
|--------|------|--------|----------|----------|
| Level 1 | 導入・単純反復 | 20問 | 1分 | 0.5, 0.25, 0.75 の基本変換のみ |
| Level 2 | 定着・パターン網羅 | 20問 | 1分30秒 | 1/8系列（0.125～0.875）、1/5系列（0.2～0.8） |

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **暗記の徹底**: 0.5=1/2、0.25=1/4、0.75=3/4 の3パターンのみ
- **同じ問題の反復OK**: エピソード記憶ではなく意味記憶（概念）を強化
- **双方向**: 小数→分数、分数→小数の両方向を混在

### 問題例

```
0.5 → □（既約分数）
1/2 → □（小数）
0.25 → □（既約分数）
1/4 → □（小数）
0.75 → □（既約分数）
3/4 → □（小数）
... （同じパターンの繰り返し）
```

### 注意事項

- ✅ 同じ問題を何度も繰り返してOK（暗記項目）
- ❌ 1/3や1/6など、循環小数は**含めない**（混乱を避ける）
- ✅ 既約分数の指定を明記（2/4ではなく1/2）

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **1/8系列の導入**: 0.125, 0.25, 0.375, 0.5, 0.625, 0.75, 0.875
- **1/5系列の導入**: 0.2, 0.4, 0.6, 0.8
- **パターン認識**: 0.125の倍数という構造を見抜く

### 問題例

```
1/8 → □（小数）        → 0.125
3/8 → □（小数）        → 0.375
5/8 → □（小数）        → 0.625
7/8 → □（小数）        → 0.875
1/5 → □（小数）        → 0.2
2/5 → □（小数）        → 0.4
0.2 → □（既約分数）    → 1/5
0.6 → □（既約分数）    → 3/5
```

### 注意事項

- ✅ 1/8系列と1/5系列をバランスよく配置
- ❌ 0.333...（1/3）や0.666...（2/3）は含めない
- ✅ Level 1の復習問題も数問混ぜる（0.5, 0.25など）

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **1より大きい数**: 1.25, 1.5, 1.75 など（帯分数・仮分数）
- **逆変換のシャッフル**: 小数→分数と分数→小数をランダムに
- **瞬発力**: 考える時間を与えない（自動化の確認）

### 問題例

```
1.25 → □（既約分数）   → 5/4
1.5 → □（既約分数）    → 3/2
1.75 → □（既約分数）   → 7/4
5/4 → □（小数）        → 1.25
3/2 → □（小数）        → 1.5
7/4 → □（小数）        → 1.75
1/8 → □（小数）        → 0.125（Level 2の復習）
0.875 → □（既約分数）  → 7/8
```

### 注意事項

- ✅ 1より大きい数を必ず含める（仮分数の理解）
- ✅ 変換方向をシャッフルして自動化を確認
- ❌ 帯分数（1と1/4）ではなく、仮分数（5/4）で答えさせる

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **計算での活用**: 小数と分数の混在計算
- **実践的な応用**: 0.5+1/4 など、どちらかに統一して計算
- **分数への変換が有利**: 0.75×8 → 3/4×8 = 6

### 問題例

```
0.5 + 1/4 = □（分数）     → 3/4
3/4 - 0.25 = □（分数）    → 1/2
0.75 × 8 = □              → 6
0.125 × 8 = □             → 1
1/4 + 0.5 = □（小数）     → 0.75
0.8 × 2.5 = □             → 2（両方を分数に）
```

### 注意事項

- ✅ 小数と分数の混在計算
- ✅ 答えの形式を指定（分数で、小数で）
- ✅ 分数に変換すると約分できる問題を選ぶ（0.75×8など）

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **逆算**: □×0.25=0.5 など、未知数を求める
- **3項計算**: 0.5+1/4-0.125 など
- **大小比較**: 0.6と5/8のどちらが大きいか
- **Sランク向け**: 最高難度の問題

### 問題例

```
□ × 0.25 = 0.5（□は？）          → 2
□ × 1/8 = 0.25（□は？）          → 2
0.5 + 1/4 - 0.125 = □（分数）    → 5/8
0.6と5/8のどちらが大きいか       → 0.6（0.6=3/5、5/8=0.625）
(0.5 + 1/4) × 8 = □              → 6
□ ÷ 0.25 = 4（□は？）            → 1
```

### 注意事項

- ✅ 逆算問題を必ず含める（□の位置を変える）
- ✅ 3項以上の複合計算
- ✅ 大小比較は通分・小数化のどちらでも解けるように

## 避けるべきパターン

### ❌ 循環小数の使用

```
NG: 1/3 → 0.333...
NG: 2/3 → 0.666...
理由: 小学生には混乱を招く。循環小数の扱いは中学数学で。
```

### ❌ 逆問題の連続配置

```
NG:
  (1) 0.5 → 1/2
  (2) 1/2 → 0.5
理由: 答えが簡単にわかってしまう。数問空けるか、別の数値を使う。
```

### ❌ 約分を忘れる

```
NG: 0.5 → 5/10（約分されていない）
OK: 0.5 → 1/2（既約分数）
理由: 必ず「既約分数」を指定する。
```

## サバイバルテストでの出題方針

### Level 1（Q1）

- 0.5→1/2 または 0.25→1/4（最も基本的な変換）

### Level 2（Q8）

- 1/8→0.125 または 3/8→0.375（1/8系列の基本）

### Level 3（Q15）

- 1.25→5/4 または 1.5→3/2（1より大きい数）

### Level 4（Q22）

- 0.5+1/4=□（分数）（計算での活用）

### Level 5（Q29）

- □×0.25=0.5（□は？）（逆算）

### セット間の多様性

- **セットA**: 0.5→1/2、1/8→0.125、1.25→5/4、0.5+1/4、□×0.25=0.5
- **セットB**: 0.25→1/4、3/8→0.375、1.5→3/2、3/4-0.25、□×1/8=0.25
- **セットC**: 0.75→3/4、5/8→0.625、1/8→0.125、1/8+0.375、0.125+□=0.5
