## 単元の目的

計算の途中で小数↔分数を変換！0.5+1/4 → 1/2+1/4のように、計算しやすい形に変える構造視力を鍛える。

### 教育的意義

- **柔軟な変換**: 小数と分数を状況に応じて使い分ける
- **計算の工夫**: 0.75×8 → 3/4×8 = 6（分数で簡単に）
- **構造視力**: どちらが計算しやすいか瞬時に判断
- **認知負荷の削減**: 最適な形式で計算

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **足し算での変換**: 0.5+1/4 → 1/2+1/4 = 3/4
- **どちらかに統一**: 小数または分数に統一して計算
- **1/8, 1/4, 1/5系列**: 基本的な分数のみ

### 問題例

```
0.5 + 1/4 → 1/2+1/4 = 3/4 = 0.75
1/2 + 0.25 → 0.5+0.25 = 0.75
0.25 + 1/2 → 0.75
0.125 + 1/8 → 1/8+1/8 = 2/8 = 0.25
0.2 + 1/5 → 1/5+1/5 = 2/5 = 0.4
```

### 注意事項

- ✅ 小数と分数のどちらに統一してもOK
- ✅ 変換の手順を明記
- ✅ 1/8, 1/4, 1/5系列に限定

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **掛け算での変換**: 0.75×8 → 3/4×8 = 6
- **分数が有利**: 約分できる場合は分数に変換
- **様々なパターン**: 様々な小数での練習

### 問題例

```
0.75 × 8 → 3/4×8 = 6
0.25 × 12 → 1/4×12 = 3
0.5 × 18 → 1/2×18 = 9
0.125 × 16 → 1/8×16 = 2
0.2 × 15 → 1/5×15 = 3
0.375 × 8 → 3/8×8 = 3
```

### 注意事項

- ✅ 分数に変換すると約分できる問題
- ✅ 計算が簡単になることを明記
- ✅ 様々な小数パターン

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **引き算**: 1-0.375 → 1-3/8 = 5/8
- **割り算**: 0.5÷0.25 → 1/2÷1/4 = 2
- **瞬発力**: 最適な変換を瞬時に判断

### 問題例

```
1 - 0.375 → 1-3/8 = 5/8 = 0.625
1 - 0.625 → 1-5/8 = 3/8 = 0.375
0.5 ÷ 0.25 → 1/2÷1/4 = 2
0.75 ÷ 0.25 → 3/4÷1/4 = 3
3/4 - 0.5 → 3/4-1/2 = 1/4
```

### 注意事項

- ✅ 引き算・割り算での変換
- ✅ 補数の知識も活用（1-0.375）
- ✅ 割り算は分数が簡単

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **両方を分数に**: 0.8×2.5 → 4/5×5/2 = 2
- **約分の発見**: 分子と分母で約分
- **実践的応用**: 複雑な小数の計算

### 問題例

```
0.8 × 2.5 → 4/5×5/2 = 2
0.6 × 1.5 → 3/5×3/2 = 9/10 = 0.9
1.2 × 0.75 → 6/5×3/4 = 18/20 = 0.9
1.6 × 0.625 → 8/5×5/8 = 1
0.125 × 16 → 1/8×16 = 2
```

### 注意事項

- ✅ 両方を分数に変換
- ✅ 約分を明示
- ✅ 実践的な応用

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **多項計算**: (0.5+1/4)×8
- **逆算**: □×0.25=0.5
- **Sランク向け**: 最高難度の問題

### 問題例

```
1.25 × 3.2 → 5/4×16/5 = 4
2.5 × 1.6 → 5/2×8/5 = 4
(0.5+1/4) × 8 → 3/4×8 = 6
(3/4-0.25) × 12 → 1/2×12 = 6
0.125 + 0.375 + 0.5 → 1/8+3/8+4/8 = 1
```

### 注意事項

- ✅ 複合計算
- ✅ 括弧の処理
- ✅ 最適な変換を見抜く

## 避けるべきパターン

### ❌ 循環小数

```
NG: 1/3 + 0.5 = 0.833...
理由: 循環小数は小学生には難しい。
```

### ❌ 複雑すぎる分数

```
NG: 7/13 × 1.2
理由: 7/13は約分できない。基本的な分数に限定。
```
