## 単元の目的

「素数もどき」（51, 57, 91, 77など）の素因数分解と倍数判定を瞬時に実行できる状態を目指す。約分の基礎となる重要スキル。

### 教育的意義

- **倍数判定法の習得**: 3の倍数（各位の和）、4の倍数（下2桁）の判定
- **素因数分解の暗記**: 91=7×13, 77=7×11, 119=7×17などの頻出パターン
- **約分への応用**: 素因数分解により最大公約数を発見
- **認知負荷の削減**: 瞬時の判定により、複雑な計算を高速化

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **基本の暗記**: 51=3×17, 57=3×19, 91=7×13, 77=7×11
- **3の倍数判定**: 各位の和が3の倍数 → 51（5+1=6）
- **7の倍数**: 91=7×13, 77=7×11は頻出のため暗記必須
- **逆問題**: 7×13=□、91÷7=□など

### 問題例

```
51を素因数分解せよ → 3×17（5+1=6で3の倍数）
57を素因数分解せよ → 3×19（5+7=12で3の倍数）
91を素因数分解せよ → 7×13（暗記必須！）
77を素因数分解せよ → 7×11（暗記必須！）
7×13 = □ → 91
91÷7 = □ → 13
```

### 注意事項

- ✅ 3の倍数判定法（各位の和）を明記
- ✅ 91=7×13, 77=7×11は「暗記必須」と強調
- ✅ 逆問題（掛け算・割り算形式）も出題
- ❌ 素数の定義は説明しない（既習前提）

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **7・11・13の組み合わせ**: 119=7×17, 133=7×19, 143=11×13, 221=13×17
- **パターン網羅**: 7, 11, 13と17, 19の組み合わせを完全網羅
- **逆問題**: 119÷17=□、11×13=□など

### 問題例

```
119を素因数分解せよ → 7×17
133を素因数分解せよ → 7×19
143を素因数分解せよ → 11×13
187を素因数分解せよ → 11×17
221を素因数分解せよ → 13×17
7×17 = □ → 119
11×13 = □ → 143
```

### 注意事項

- ✅ 7・11・13の倍数を網羅的に出題
- ✅ 逆問題で定着を確認
- ✅ 3²×13=117など、平方数を含むパターンも出題

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **素数判定（○×）**: 87, 117, 161は素数か？
- **倍数判定**: 3の倍数（各位の和）、4の倍数（下2桁）
- **混在問題**: 素因数分解と判定問題をシャッフル
- **瞬発力**: 考える時間を与えない（自動化の確認）

### 問題例

```
372は3の倍数か？（○×） → ○（3+7+2=12）
816は4の倍数か？（○×） → ○（下2桁16÷4=4）
87は素数か？（○×） → ×（87=3×29）
161は素数か？（○×） → ×（161=7×23）
408は何の倍数か？（3, 4, 両方） → 両方
```

### 注意事項

- ✅ 3の倍数判定（各位の和が3の倍数）
- ✅ 4の倍数判定（下2桁が4の倍数）
- ✅ ○×判定のパターンを混在させる（○と×の偏りを避ける）

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **約分への応用**: 84/132のGCD（最大公約数）を求める
- **実践的約分**: 91/119, 51/85など、素因数分解を利用
- **構造視力**: 分子と分母の共通因数を瞬時に発見

### 問題例

```
84/132の約分：GCDは？ → 12（84=12×7、132=12×11）
91/119の約分：GCDは？ → 7（91=7×13、119=7×17）
51/85を約分せよ → 3/5（51=3×17、85=5×17）
77/91を約分せよ → 11/13（77=7×11、91=7×13）
```

### 注意事項

- ✅ GCD（最大公約数）を明示的に求める問題
- ✅ 素因数分解から約分を導く
- ✅ 約分後の既約分数を答えとする

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **複雑な約分**: 87/116（GCD=29）など、見抜きにくい共通因数
- **平方数の発見**: 323=17²など
- **Sランク向け**: 最高難度の問題

### 問題例

```
87/116を約分せよ → 3/4（87=3×29、116=4×29）
119/136を約分せよ → 7/8（119=7×17、136=8×17）
57/133を約分せよ → 3/7（57=3×19、133=7×19）
286は何と何の積か？（7×□） → 7×41
323は何の平方数か？（□²） → 17²
```

### 注意事項

- ✅ 共通因数が19, 23, 29など大きな素数
- ✅ 平方数の発見（17²=289、19²=361など）
- ✅ 逆算問題を含める

## 避けるべきパターン

### ❌ 大きすぎる数

```
NG: 1001 = 7×11×13
理由: 3桁以上の素因数分解は難易度が高すぎる。
```

### ❌ 循環小数の扱い

```
NG: 1/7 = 0.142857...
理由: 本単元は整数の素因数分解。小数は範囲外。
```

### ❌ 素数の判定（大きな数）

```
NG: 997は素数か？
理由: 大きな素数の判定は試し割りが大変。2桁に限定。
```
