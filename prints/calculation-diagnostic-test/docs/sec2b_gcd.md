## 単元の目的

99×36 = (100-1)×36、101×24 = (100+1)×24など、100に近い数を分解して計算する技法を習得する。

### 教育的意義

- **構造の発見**: 101=100+1、99=100-1と見る視力
- **計算の工夫**: 大きな掛け算を簡単な計算に変換
- **代数的思考**: (100±n)×aのパターン認識
- **認知負荷の削減**: 暗算の範囲内で計算可能に

---

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **101=100+1**: 101×24 → (100+1)×24 = 2400+24 = 2424
- **パターンの暗記**: 101×nは必ず100n+nになる
- **反復の徹底**: 同じパターンを繰り返し練習

### 問題例

```
101×12 → (100+1)×12 = 1200+12 = 1212
101×24 → (100+1)×24 = 2400+24 = 2424
101×36 → 3600+36 = 3636
101×42 → 4200+42 = 4242
101×53 → 5300+53 = 5353
```

### 注意事項

- ✅ 101=100+1の分解を明記
- ✅ 答えのパターン（abab形式）
- ✅ 同じ問題を繰り返してOK

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **99=100-1**: 99×36 → (100-1)×36 = 3600-36 = 3564
- **102=100+2**: 102×25 → (100+2)×25 = 2500+50 = 2550
- **98=100-2**: 98×32 → (100-2)×32 = 3200-64 = 3136
- **パターン網羅**: 様々な数での練習

### 問題例

```
99×36 → (100-1)×36 = 3600-36 = 3564
99×47 → 4700-47 = 4653
102×25 → (100+2)×25 = 2500+50 = 2550
102×38 → 3800+76 = 3876
98×32 → (100-2)×32 = 3200-64 = 3136
98×45 → 4500-90 = 4410
```

### 注意事項

- ✅ 99, 98（引き算）と102（足し算）を混在
- ✅ 引き算の注意（マイナスの計算）
- ✅ 様々な数で練習

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **103, 97など**: 103×25 → (100+3)×25 = 2500+75 = 2575
- **104, 96, 105, 95**: 様々な100近傍の数
- **瞬発力**: 100との差を瞬時に見抜く

### 問題例

```
103×25 → (100+3)×25 = 2500+75 = 2575
97×34 → (100-3)×34 = 3400-102 = 3298
104×23 → (100+4)×23 = 2300+92 = 2392
96×35 → (100-4)×35 = 3500-140 = 3360
105×18 → (100+5)×18 = 1800+90 = 1890
95×24 → (100-5)×24 = 2400-120 = 2280
```

### 注意事項

- ✅ 100との差（±3, ±4, ±5）を見抜く
- ✅ 引き算の計算（3×34=102など）に注意
- ✅ 足し算と引き算を混在

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **実践的応用**: 98×47, 102×78など
- **複雑な計算**: 引き算が2倍（98×63 → 6300-126）
- **構造視力**: 瞬時に100との差を見抜く

### 問題例

```
98×47 → (100-2)×47 = 4700-94 = 4606
98×63 → 6300-126 = 6174
102×49 → (100+2)×49 = 4900+98 = 4998
102×78 → 7800+156 = 7956
97×43 → (100-3)×43 = 4300-129 = 4171
103×52 → (100+3)×52 = 5200+156 = 5356
```

### 注意事項

- ✅ 実践的な数での練習
- ✅ 引き算の計算ミスに注意
- ✅ 様々なパターン

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **両方とも100に近い**: 99×98 → (100-1)×(100-2)
- **展開の公式**: (a+b)(c+d) = ac+ad+bc+bd
- **Sランク向け**: 最高難度の問題

### 問題例

```
99×98 → (100-1)×(100-2) = 10000-200-100+2 = 9702
101×102 → (100+1)×(100+2) = 10000+200+100+2 = 10302
99×101 → (100-1)×(100+1) = 10000-1 = 9999（和と差の積）
98×102 → (100-2)×(100+2) = 10000-4 = 9996
99×99 → (100-1)² = 10000-200+1 = 9801
101×101 → (100+1)² = 10000+200+1 = 10201
```

### 注意事項

- ✅ 展開の公式（和と差の積、平方の展開）
- ✅ 符号に注意（±の計算）
- ✅ 暗算の範囲内で計算

## 避けるべきパターン

### ❌ 100から遠すぎる数

```
NG: 85×76
理由: 100から15も離れている。工夫の効果が薄い。
```

### ❌ 小数の混在

```
NG: 99.5×24
理由: 本単元は整数に限定。小数は別単元。
```
