# 【単元B】四角数・三角数 設計書

**ファイル**: `sec1b_squares.js`
**ステージ**: 第1部：反射神経
**単元ID**: `sec1b`

---

## 単元の目的

四角数（平方数）と三角数を完全暗記し、瞬時に答えられる状態（System 1：直感）を目指す。

---

## レベル構成

| レベル | 名称 | 左右の構成 | 主な内容 |
|--------|------|:---:|----------|
| Level 1 & 2 | | 四角数和(9問) vs 三角数(13問) | 11-19平方数 / 1-10和(6問) ＋ 1-10逆算(7問) |
| Level 3 & 4 | | 四角数逆算(ランダム) vs 三角数和 | 11-19逆算(シャッフル) / 11-20和 |
| Level 5 | | 小数平方数 vs 三角数逆算 | 1.1-1.9小数 / 11-20逆算 |
