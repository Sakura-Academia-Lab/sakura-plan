## 単元の目的

18×5÷6 → 最初に約分！共通因数を見つけて消してから計算する構造視力を鍛える。

### 教育的意義

- **約分の前倒し**: 掛けてから割るのではなく、先に約分
- **共通因数の発見**: 18と6の共通因数は6
- **計算の簡略化**: 18×5÷6 → 3×5 = 15
- **認知負荷の削減**: 大きな数にならないよう先に約分

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **基本形**: 18×5÷6 → 18と6で約分 → 3×5 = 15
- **共通因数の発見**: 18と6の最大公約数は6
- **約分の手順**: 先に約分してから掛け算

### 問題例

```
18×5÷6 → 3×5 = 15（18と6を約分）
12×7÷4 → 3×7 = 21（12と4を約分）
15×8÷5 → 3×8 = 24（15と5を約分）
20×9÷4 → 5×9 = 45（20と4を約分）
24×5÷8 → 3×5 = 15（24と8を約分）
```

### 注意事項

- ✅ 約分の手順を明記
- ✅ 共通因数を見つける練習
- ✅ 先に約分することを強調

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **様々な数**: 様々な共通因数での練習
- **複数の約分**: 24×15÷18など複数箇所で約分
- **パターン網羅**: 様々なパターンでの練習

### 問題例

```
24×15÷18 → 4×5 = 20（24と18が6、15と18が3）
36×10÷12 → 3×10 = 30（36と12で約分）
45×8÷15 → 3×8 = 24（45と15で約分）
32×21÷14 → 16×3 = 48（21と14で約分）
```

### 注意事項

- ✅ 複数箇所での約分
- ✅ 約分の順序は自由
- ✅ 様々なパターン

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **分数形式**: 2/3 × 9/4 → 分子と分母で約分
- **約分の発見**: 2と4、9と3で約分
- **瞬発力**: 瞬時に約分箇所を見つける

### 問題例

```
2/3 × 9/4 → (2×9)/(3×4) = 18/12 = 3/2
4/5 × 15/8 → (4×15)/(5×8) = 60/40 = 3/2
6/7 × 14/9 → (6×14)/(7×9) = 84/63 = 4/3
8/9 × 27/16 → (8×27)/(9×16) = 216/144 = 3/2
```

### 注意事項

- ✅ 分数の掛け算での約分
- ✅ 分子と分母をクロスで約分
- ✅ 最終的に既約分数にする

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **複雑な形式**: (18×25)/(15×12)
- **多段階の約分**: 複数箇所で約分
- **実践的応用**: 実際の計算での活用

### 問題例

```
(18×25)/(15×12) → (3×5)/(3×2) = 5/2 = 2.5
(24×35)/(28×30) → (4×5)/(4×5) = 1
(32×45)/(36×40) → (4×9)/(9×4) = 1
(42×20)/(35×24) → (6×5)/(5×6) = 1
```

### 注意事項

- ✅ 分子と分母の複数項
- ✅ 対称性に注目（答えが1になる）
- ✅ 実践的な応用

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **3項以上**: (12×35×48)/(14×40×36)
- **複雑な約分**: 複数箇所で約分
- **Sランク向け**: 最高難度の問題

### 問題例

```
(12×35×48)/(14×40×36) → 4/3
(15×42×64)/(21×56×60) → 4/5
2/3 × 9/4 × 8/15 → 4/5
5/6 × 12/25 × 15/8 → 3/4
```

### 注意事項

- ✅ 3項以上の複雑な計算
- ✅ 段階的に約分
- ✅ 最終的に既約分数にする

## 避けるべきパターン

### ❌ 約分できない問題

```
NG: 17×23÷19
理由: 共通因数がない。約分の練習にならない。
```

### ❌ 複雑すぎる数

```
NG: 143×187÷221
理由: 大きすぎる。2桁までに限定。
```
