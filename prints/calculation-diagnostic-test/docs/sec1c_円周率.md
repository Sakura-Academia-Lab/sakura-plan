## 単元の目的

3.14×2, 3.14×3, ...などの「円周率の段」を瞬時に計算できる状態を目指す。円の面積・円周の計算で頻出するため、暗記必須。

### 教育的意義

- **暗記項目**: 3.14×2～9の基本計算を長期記憶化
- **実用性**: 円の面積（πr²）、円周（2πr）の計算で日常的に使用
- **計算の工夫**: 3.14×10=31.4（小数点移動）、3.14×0.5=1.57（÷2）など
- **認知負荷の削減**: 暗記により、円の問題を瞬時に解答可能

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **完全暗記**: 3.14×2=6.28, 3.14×3=9.42, ..., 3.14×9=28.26
- **反復の徹底**: 同じ問題を繰り返し出題してOK
- **計算方法**: 3.14×2 → 3×2=6, 0.14×2=0.28 → 6.28

### 問題例

```
3.14 × 2 → 6.28
3.14 × 3 → 9.42
3.14 × 4 → 12.56
3.14 × 5 → 15.7
3.14 × 6 → 18.84
3.14 × 7 → 21.98
3.14 × 8 → 25.12
3.14 × 9 → 28.26
```

### 注意事項

- ✅ 同じ問題を何度も繰り返してOK（暗記項目）
- ✅ 3.14×5=15.7（15.70ではなく）と表記
- ❌ 3.14×1は含めない（自明）

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **10の倍数**: 3.14×10=31.4, 3.14×100=314（小数点移動）
- **20, 30, 40...**: 3.14×20=62.8（=3.14×2×10）
- **11, 12の段**: 3.14×11=34.54（=31.4+3.14）

### 問題例

```
3.14 × 10 → 31.4（小数点1つ右）
3.14 × 20 → 62.8
3.14 × 100 → 314（小数点2つ右）
3.14 × 11 → 34.54（31.4+3.14）
3.14 × 12 → 37.68（31.4+6.28）
```

### 注意事項

- ✅ 小数点移動のルールを明記
- ✅ 11, 12の段は10の段との組み合わせ
- ✅ Level 1の復習問題も混ぜる

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **小数倍**: 3.14×0.5=1.57（÷2）、3.14×1.5=4.71（3.14+1.57）
- **大きな数**: 3.14×200, 3.14×500など
- **瞬発力**: 考える時間を与えない（自動化の確認）

### 問題例

```
3.14 × 0.1 → 0.314（小数点1つ左）
3.14 × 0.5 → 1.57（3.14÷2）
3.14 × 1.5 → 4.71（3.14+1.57）
3.14 × 200 → 628
3.14 × 500 → 1570
```

### 注意事項

- ✅ 小数と大きな数を混在させる
- ✅ 0.5倍は「÷2」と明記
- ✅ 1.5, 2.5は基本値との和で計算

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **円の面積**: πr² = 3.14×r²
- **円の円周**: 2πr = 2×3.14×r
- **実践的応用**: 半径・直径から面積・円周を求める

### 問題例

```
半径2cmの円の面積 → πr² = 3.14×4 = 12.56cm²
半径3cmの円の円周 → 2πr = 2×3.14×3 = 18.84cm
直径4cmの円の面積 → 半径2cm → 3.14×4 = 12.56cm²
半径10cmの円の面積 → 3.14×100 = 314cm²
```

### 注意事項

- ✅ 面積の公式（πr²）を明記
- ✅ 円周の公式（2πrまたはπd）を使い分ける
- ✅ 直径から半径への変換を確認

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **逆算**: 3.14×□=15.7 → □=5
- **面積からの逆算**: 面積12.56cm² → 半径2cm
- **分配法則**: 3.14×8-3.14×3 = 3.14×(8-3) = 3.14×5

### 問題例

```
3.14 × □ = 15.7 → □ = 5
面積が12.56cm²の円の半径 → πr²=12.56 → r²=4 → r=2cm
円周が18.84cmの円の半径 → 2πr=18.84 → r=3cm
3.14 × 8 - 3.14 × 3 = 3.14×5 = 15.7（分配法則）
```

### 注意事項

- ✅ 逆算問題を必ず含める
- ✅ 面積・円周からの逆算（r²やrを求める）
- ✅ 分配法則の応用

## 避けるべきパターン

### ❌ 円周率を3や3.1で計算

```
NG: 3×5 = 15（円周率を3で計算）
理由: 本単元は3.14を使った計算の訓練。
```

### ❌ 複雑な小数

```
NG: 3.14 × 1.23 = 3.8622
理由: 計算が複雑すぎる。整数か0.5の倍数に限定。
```

### ❌ 球や円柱の問題

```
NG: 半径3cmの球の体積 → (4/3)πr³
理由: 本単元は平面図形（円）に限定。立体は範囲外。
```
