## 単元の目的

a×b + a×c = a×(b+c) の発見！共通因数を見つけて括り出す構造視力を鍛える。

### 教育的意義

- **構造視力の育成**: 共通因数aを瞬時に発見する力
- **計算の工夫**: 34×13+34×87 → 34×(13+87) = 34×100 = 3400
- **代数的思考**: 文字式の理解への布石
- **認知負荷の削減**: 複雑な計算を簡略化

## Level 1：導入・単純反復（20問 / 1分）

### 設計意図

- **(b+c)=100**: 34×13+34×87 → 34×(13+87) = 34×100
- **共通因数の発見**: aを見つける目を鍛える
- **反復の徹底**: 同じパターンを繰り返し練習

### 問題例

```
34×13 + 34×87 → 34×(13+87) = 34×100 = 3400
27×15 + 27×85 → 27×(15+85) = 27×100 = 2700
56×32 + 56×68 → 56×(32+68) = 56×100 = 5600
42×47 + 42×53 → 42×(47+53) = 42×100 = 4200
```

### 注意事項

- ✅ (b+c)=100になる問題を中心に
- ✅ 共通因数を明示的に示す
- ✅ 括り出しの手順を明記

---

## Level 2：定着・パターン網羅（20問 / 1分30秒）

### 設計意図

- **様々な数**: (b+c)が100以外のパターン
- **パターン網羅**: 様々な共通因数での練習
- **定着の確認**: Level 1の復習も混ぜる

### 問題例

```
28×37 + 28×63 → 28×(37+63) = 28×100 = 2800
15×24 + 15×76 → 15×(24+76) = 15×100 = 1500
48×51 + 48×49 → 48×(51+49) = 48×100 = 4800
```

### 注意事項

- ✅ (b+c)=100のパターンを継続
- ✅ 様々な共通因数a
- ✅ Level 1の復習問題も混ぜる

---

## Level 3：壁・罠・瞬発力（15問 / 2分）

### 設計意図

- **引き算の分配**: a×b - a×c = a×(b-c)
- **瞬発力**: 足し算と引き算を混在させる
- **構造視力**: 引き算でも共通因数を見つける

### 問題例

```
58×23 - 58×13 → 58×(23-13) = 58×10 = 580
93×177 - 93×77 → 93×(177-77) = 93×100 = 9300
125×17 - 125×9 → 125×(17-9) = 125×8 = 1000
47×56 - 47×46 → 47×(56-46) = 47×10 = 470
```

### 注意事項

- ✅ 引き算の分配法則
- ✅ (b-c)が10, 50, 100など計算しやすい数
- ✅ 足し算と引き算を混在させる

---

## Level 4：応用・実践投入（10問 / 2分）

### 設計意図

- **3項以上**: a×b + a×c + a×d
- **足し引き混在**: a×b + a×c - a×d
- **実践的応用**: 複雑な計算での活用

### 問題例

```
12×7 + 12×8 + 12×5 → 12×(7+8+5) = 12×20 = 240
25×3 + 25×5 + 25×2 → 25×(3+5+2) = 25×10 = 250
36×6 - 36×4 + 36×8 → 36×(6-4+8) = 36×10 = 360
45×5 + 45×3 - 45×2 → 45×(5+3-2) = 45×6 = 270
```

### 注意事項

- ✅ 3項以上の分配
- ✅ 足し算と引き算の混在
- ✅ (合計)が簡単な数になるよう設計

---

## Level 5：達人・複合問題（10問 / 3分）

### 設計意図

- **隠れた1**: 999×37+37 → 37×(999+1) = 37×1000
- **構造の発見**: 37が共通因数（37×1と見る）
- **Sランク向け**: 最高難度の問題

### 問題例

```
999×37 + 37 → 37×(999+1) = 37×1000 = 37000
48×99 + 48 → 48×(99+1) = 48×100 = 4800
125×79 + 125 → 125×(79+1) = 125×80 = 10000
84×24 + 84×25 + 84 → 84×(24+25+1) = 84×50 = 4200
```

### 注意事項

- ✅ 「隠れた1」を見つける（37=37×1）
- ✅ 99, 999など特殊な数
- ✅ 複数項と隠れた1の組み合わせ

---

## 避けるべきパターン

### ❌ 共通因数がない問題

```
NG: 34×13 + 56×87
理由: 共通因数がない。分配法則が使えない。
```

### ❌ 複雑すぎる計算

```
NG: 123×456 + 123×789
理由: 桁数が多すぎる。2桁までに限定。
```

### ❌ 小数・分数の混在

```
NG: 3.14×7 + 3.14×3
理由: 本単元は整数に限定。小数は別単元。
```
